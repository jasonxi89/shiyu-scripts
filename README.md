## 寿喜锅自动生产线

厨师发需求单 → 采集号抢单采集/购买 → 宠物邮件发给厨师 → 厨师制造

| 脚本 | 角色 | 功能 |
|------|------|------|
| 厨师_寿喜锅 | 厨师 | 站医院制造+补魔+发需求单 |
| 采集_葱 | 采集号 | 伊尔村→芙蕾雅(660,235) |
| 采集_酱油 | 采集号 | 法兰东门→芙蕾雅(501,161) |
| 采集_盐 | 采集号 | 亚留特村→芙蕾雅(712,172) |
| 采集_牛肉 | 采集号 | 哥拉尔→库鲁克斯岛(347,406) |
| 买糖_砂糖 | 买糖号 | 维诺亚村修葛买糖 |

### 需求单机制
- 厨师检测材料 < 120 → 发需求单(每槽20, 最多5槽)
- 采集号扫描需求1~5 → 找可用宠物(1~5号) → 认领+宠邮发货
- 内存变量: `厨师`=角色名, `材料_需求1`~`材料_需求5`

### 特性
- 从任意位置启动续跑
- 没蓝回最近村子补(不回法兰)
- 背包满原地等需求发宠邮
- 每采20个检查一次需求
- 宠邮自动扫描1~5号可用宠物
